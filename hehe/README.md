# Module-starter

```sh
git clone https://github.com/rognstadragnar/module-starter.git
rm -rf .git
git init
# Update package.json and README to make it your own
npm i
```

or with degit:

```sh
degit rognstadragnar/module-starter my-module
cd my-module
# Update package.json and README to make it your own
npm i
```
